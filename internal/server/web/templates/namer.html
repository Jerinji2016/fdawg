Platform Information

Android

App name is configured in android/app/src/main/AndroidManifest.xml

Property: android:label

iOS

App name is configured in ios/Runner/Info.plist

Properties: CFBundleDisplayName and CFBundleName

macOS

App name is configured in macos/Runner/Configs/AppInfo.xcconfig

Property: PRODUCT_NAME

Linux

App name is configured in linux/CMakeLists.txt

Property: BINARY_NAME

Windows

App name is configured in windows/CMakeLists.txt

Properties: project() and BINARY_NAME

Web

App name is configured in web/manifest.json and web/index.html

Properties: name, short_name, and <title>
