Universal Name

Platform-Specific Names

Platform Information

Android
Configuration File
android/app/src/main/AndroidManifest.xml
Property
android:label
The app name appears in the app launcher and system settings.
iOS
Configuration File
ios/Runner/Info.plist
Properties
CFBundleDisplayName CFBundleName
CFBundleDisplayName is shown on the home screen, CFBundleName is the internal identifier.
macOS
Configuration File
macos/Runner/Configs/AppInfo.xcconfig
Property
PRODUCT_NAME
The app name appears in the dock, menu bar, and Applications folder.
Linux
Configuration File
linux/CMakeLists.txt
Property
BINARY_NAME
The binary name used for the executable file and desktop entry.
Windows
Configuration File
windows/CMakeLists.txt
Properties
project() BINARY_NAME
The project name and binary name for the Windows executable.
Web
Configuration Files
web/manifest.json web/index.html
Properties
name, short_name <title>
The app name appears in the browser tab, PWA installation, and web app manifest.
