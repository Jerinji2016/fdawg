Bundle ID Guidelines

Format
Reverse domain notation (e.g., com.company.appname)
Valid Characters
Letters (a-z, A-Z), numbers (0-9), dots (.), hyphens (-), underscores (_)
Rules
• Must contain at least one dot
• Cannot start or end with a dot
• No consecutive dots
• Segments cannot start with numbers
• 3-255 characters long
Examples
✓ com.mycompany.myapp
✓ org.example.flutter-app
✗ myapp (no domain)
✗ com..myapp (consecutive dots)
✗ com.123company.app (segment starts with number)
